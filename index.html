Halo, Saya Radithya 👋

Perjalanan saya di dunia coding dimulai saat kelas 5 SD, atau menjelang kenaikan ke kelas 6. Saat itu saya sedang sangat suka bermain game, terutama game bergenre simulator buatan developer lokal Andy Lukito (Akhir Pekan Studio). Dari situ muncul ketertarikan baru: saya tidak ingin hanya menjadi penikmat game, tetapi juga ingin menjadi pembuatnya. Saya juga terinspirasi dari beliau untuk mulai terjun ke dunia development.

Namun, saat itu saya belum memiliki laptop, yang cukup menghambat untuk belajar game development. Akhirnya saya mencoba beralih ke web development. Awalnya hanya iseng menggunakan template HTML, CSS, dan JavaScript untuk membuat kartu ucapan dan proyek sederhana lainnya. Dari situ, minat saya mulai berkembang dengan cukup pesat.

Proyek pertama saya adalah membuat website tentang batik Neo Tangsel dalam kegiatan P5 (Proyek Penguatan Profil Pelajar Pancasila) saat kelas 7 SMP. Dari proyek ini, saya mulai memahami dasar-dasar pembuatan website secara lebih serius.

Selanjutnya, saya membuat proyek kedua berupa website top up game dan voucher bernama GARUDANEXTERA, yang saya mulai pada awal kelas 8 SMP. Menariknya, saya mengerjakannya hanya menggunakan HP dan platform Replit. Setelah saya memiliki laptop, proyek tersebut saya lanjutkan.

Sayangnya, pada awal kelas 9 SMP, proyek GARUDANEXTERA mengalami kegagalan. Saya kehilangan seluruh database MySQL, dan banyak error pada bagian PHP. Hingga saat ini, saya belum memiliki niat untuk memperbaikinya... tetapi mungkin suatu saat nanti akan saya bangun kembali dengan versi yang lebih baik. 🙂

Proyek ketiga saya adalah tugas akhir SMP pada mata pelajaran Informatika, yaitu membuat game berbasis web menggunakan konsep RNG (Random Number Generator), dengan mengadaptasi permainan kartu UNO. Dari proyek ini, saya belajar tentang logika game, sistem random, dan interaksi user dalam aplikasi berbasis web.

Selain itu, saya juga memiliki ketertarikan di bidang cyber security. Namun, untuk saat ini saya memilih untuk lebih fokus terlebih dahulu pada web development dan game development sebagai fondasi utama kemampuan saya.

Saya memiliki mimpi untuk dapat bekerja di lembaga-lembaga besar, seperti NASA, dan berkontribusi dalam pengembangan teknologi yang berdampak luas, baik di bidang software, sistem, maupun inovasi digital lainnya.

Saat ini, saya terus belajar dan mengembangkan kemampuan di bidang HTML, CSS, JavaScript, serta mulai mendalami C# untuk game development. Saya memiliki tujuan untuk suatu saat dapat menciptakan karya yang tidak hanya menarik, tetapi juga memiliki dampak nyata.
